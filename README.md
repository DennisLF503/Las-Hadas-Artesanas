# Las Hadas Artesanas

Sitio web realizado para unos pequeños emprendedores de Artesanias llamados *Las Hadas Artesanas*
ellos realizan trabajos de Crochet, Velas Aromaticas y Flores Eternas.

# Descripcion del proyecto

El proyecto busca resolver la falta de presencia tecnologica en el emprendimiento, colocando el catálogo 
de todos los productos, la información de los emprendedores y un formulario donde podrá realizar el usuario su pedido  
el cualquier momento
